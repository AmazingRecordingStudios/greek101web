A read-along audiobook for Ancient Greek

with real quotes from Homer to Plutarch

(produced by Amazing Recording Studios)

This project is intended as a collection of quotes from classic Ancient Greek authors that you can hear spoken in the original Ancient Greek. For each one there is also a translation and some linguistic notes.
